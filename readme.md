# Project-Expense-Manager-Backend-Parser

#### Python virtual env:

How to create a virtual env:  
`python3 -m venv venv_dir`

How to activate it:  
`source ./venv/bin/activate`

How to deactivate it:  
`deactivate`

How to set Jupyter in VSC
